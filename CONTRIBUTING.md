# Contributing Guidelines

Thank you for your interest in contributing to the **Green and Sustainable Energy Review** project!

## 📋 How to Contribute

### Reporting Issues

If you find errors, outdated information, or have suggestions for improvement:

1. **Check existing issues** to avoid duplicates
2. **Open a new issue** with a clear title and description
3. Include relevant section numbers and page references

### Suggesting Updates

This is a review paper repository. Contributions may include:

- Corrections to factual information
- Updates to statistics and data (with sources)
- Additional references and citations
- Translation contributions
- Accessibility improvements

### Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-update`)
3. Make your changes with clear commit messages
4. Submit a pull request with a detailed description

## 📚 Citation Standards

When adding new references, please follow APA 7th edition format:

```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Journal, volume(issue), page range. https://doi.org/xxxxx
```

## ⚖️ Code of Conduct

- Be respectful and constructive
- Focus on the content, not the person
- Provide evidence for factual claims
- Acknowledge sources appropriately

## 📧 Contact

For major inquiries, please open an issue or contact the author.
